# OpenSnapCode

**Open Snap Code** is a free open source code editor written in Python using the TKinter library.

## Startup Requirements

For **Open Snap Code** to work, the following components are required:

- Python 3.10 (recommended)
- pyyaml 6.0.1
- TKinter
- re
- ctypes
